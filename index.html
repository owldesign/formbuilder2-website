on Craft CMS

Working with core Craft functionality, you will get seamless experience building your forms for your front-end templates.

  • Uses Craft CMS Functions

    By using Craft CMS core functionalities we are able to keep updated witht the latest version of Craft.

  • Craft Fields

    Take advantage of Craft's own Field creation tool to populate your forms with fields.

  • Database

    Each form submission will be stored in database for easy backups.

View, Export & Import

Each submission gets stored to a database. You can preview submission data via FormBuilder2 admin. Because each submission will belong to a form you can easily filter which submissions you want to see.

You can import forms from another website or export them. If you need to export your form submission you have the option to export all or filter by a form.
